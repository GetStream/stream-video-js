The SDK's default implementation of device settings components consists of audio and video device selectors. These are:

- `DeviceSelectorVideo`
- `DeviceSelectorAudioInput`
- `DeviceSelectorAudioOutput`

There is also a composing component `DeviceSettings` putting these together.

## General usage

```tsx
import { Call, DeviceSettings, StreamCall } from '@stream-io/video-react-sdk';

const MyDevicePanel = () => {
  let call: Call;

  return (
    <StreamCall call={call}>
      <DeviceSettings />
    </StreamCall>
  );
};
```

Or use the components individually:

```tsx
import {
  Call,
  DeviceSelectorAudioInput,
  DeviceSelectorVideo,
  StreamCall,
} from '@stream-io/video-react-sdk';

const MyDevicePanel = () => {
  let call: Call;

  return (
    <StreamCall call={call}>
      <DeviceSelectorVideo />
      <DeviceSelectorAudioInput />
    </StreamCall>
  );
};
```

## Props

### DeviceSettings

None

### DeviceSelectorVideo

### `title`

| Type                    | Default             |
| ----------------------- | ------------------- |
| `string` \| `undefined` | `'Select a Camera'` |

String displayed at the top of the selector component.

### DeviceSelectorAudioInput

### `title`

| Type                    | Default          |
| ----------------------- | ---------------- |
| `string` \| `undefined` | `'Select a Mic'` |

String displayed at the top of the selector component.

### DeviceSelectorAudioOutput

### `title`

| Type                    | Default             |
| ----------------------- | ------------------- |
| `string` \| `undefined` | `'Select Speakers'` |

String displayed at the top of the selector component.

## Customization

If you want to create your own component, you can make use of [`Devices Management API`](../../../guides/camera-and-microphone) to list and select devices.
