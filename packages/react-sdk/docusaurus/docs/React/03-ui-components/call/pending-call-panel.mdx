The component [`PendingCallPanel`](https://github.com/GetStream/stream-video-js/tree/main/packages/react-sdk/src/components/PendingCallPanel) is designed to represent incoming or outgoing [ring call](../../../guides/joining-and-creating-calls/#ring-call). The default implementation displays:

- call member avatars
- call connection status
- pending call control buttons to accept or cancel the call

## Pending call controls

The `PendingCallPanel` component features [`CancelCallButton`](../call-controls#cancelcallbutton) to reject resp. cancel the pending call and in case of incoming call panel, also [`AcceptCallButton`](../call-controls#acceptcallbutton) is displayed.

## Props PendingCallPanel

### `includeSelf`

| Type                     | Default |
| ------------------------ | ------- |
| `boolean` \| `undefined` | `false` |

Whether to include the current user in the list of members to show.

### `totalMembersToShow`

| Type                    | Default |
| ----------------------- | ------- |
| `number` \| `undefined` | `3`     |

The maximum number of members to show.

## Props PendingCallControls

None
