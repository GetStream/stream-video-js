title: Installation
---

The installation of the packages for the Stream Video React SDK can be either done with `yarn` or `npm`.

### Install the React SDK

For `yarn`:

```
yarn add @stream-io/video-react-sdk
```

For `npm`:

```
npm install @stream-io/video-react-sdk
```

### Install the low-level client

For `yarn`:

```
yarn add @stream-io/video-client
```

For `npm`:

```
npm install @stream-io/video-client
```
