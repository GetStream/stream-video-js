Component to customize the bottom view controls at the host's live stream.

| Type                          | Default Value                                                                                                                                                                        |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `ComponentType`\| `undefined` | [`HostLiveStreamControls`](https://github.com/GetStream/stream-video-js/blob/main/packages/react-native-sdk/src/components/Livestream/LivestreamControls/HostLivestreamControls.tsx) |

:::note
To optionally add screensharing option to livestream, you can add the exported [`LivestreamScreenShareButton`](https://github.com/GetStream/stream-video-js/blob/main/packages/react-native-sdk/src/components/Livestream/LivestreamControls/LivestreamScreenShareButton.tsx) from the SDK in your own `HostLiveStreamControls` component.
:::
